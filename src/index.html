Deformat

Paste your text below and hit Deformat to remove line breaks and extra whitespace.

No more manually removing each line break when pasting large chunks of text from PDF books. This tool also cleans up any messed up formatting that involves whitespace.

Tips

  • The following characters are removed: \r\n, \r, \n, and extra single space characters.
  • Click the Deformat button to remove whitespace.
  • Use Ctrl+A to select all text.
  • Use the Clear button to remove all text.
